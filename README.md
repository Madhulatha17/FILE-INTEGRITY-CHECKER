## Description

This project implements a File Integrity Checker using Python and the hashlib library. The tool monitors files by generating SHA-256 hash values and comparing them to detect modifications.

## Features

* SHA-256 hash generation
* File integrity verification
* Modification detection
* Command-line interface

## Technologies Used

* Python 3
* hashlib
* os

## How to Run

```bash
python file_integrity_checker.py
```

## Sample Input

sample_file.txt

## Expected Output

* File Integrity Maintained
* File Has Been Modified

## Deliverable

A Python script that monitors file changes by calculating and comparing hash values.
